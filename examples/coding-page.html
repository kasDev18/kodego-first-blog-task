When it comes to Programming, does it matter to choose a language to make you successful?

I wouldn't say not to choose your preferred language, it is based from my personal experience. Well, ofcourse you need to know first what you really want. If you wanna become developer for games,then go for C++ or language for game.

Actually, I always read some confession from experienced programmer after working hours. I've known so may things like what is the highest paid language, easiest one, and indemand all over the world. But the thing here is, would the language dictate for successfullness?

The answer is NOT, atleast based on what I read most of the time. If you start with the language that you don't really preferred ,eventually you will like it. One thing is for sure to remember, that you need to cope up with the changes as it is very constant in IT world. So if you are updated, then you're the winner!
